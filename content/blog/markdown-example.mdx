## 代码示例

行内代码：`const greeting = "Hello, World!";`

代码块：

```javascript
function sayHello(name) {
  console.log(`Hello, ${name}!`);
}

sayHello("用户");
```

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
