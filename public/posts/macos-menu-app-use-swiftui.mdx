## MenuExtra

使用 MenuExtra 可以创建一个菜单栏应用。

```swift
@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
        .menuExtra("MyApp", image: "AppIcon") {
            MenuExtra()
        }
    }
}
```
