## Tailwind CSS 的优势

Tailwind CSS 通过原子化 CSS 类名的方式，让我们能够快速构建现代化的用户界面，而无需编写自定义 CSS。

## 常用技巧

### 1. 响应式设计

```html
<div class="w-full md:w-1/2 lg:w-1/3">
  <!-- 内容会随着屏幕尺寸变化而改变宽度 -->
</div>
```

### 2. 暗色模式

```html
<div class="bg-white dark:bg-gray-800 text-black dark:text-white">
  <!-- 支持暗色模式的内容 -->
</div>
```

### 3. 悬停状态

```html
<button class="bg-blue-500 hover:bg-blue-600 transition-colors">
  点击我
</button>
```

## 自定义配置

### 扩展主题

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#1a73e8',
        secondary: '#4285f4',
      },
      spacing: {
        '128': '32rem',
      },
    },
  },
}
```

## 性能优化建议

1. 使用 JIT 模式
2. 合理使用 @apply 指令
3. 提取常用组件样式
4. 使用 purge 选项清除未使用的样式
