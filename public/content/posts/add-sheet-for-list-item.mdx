## Use id to specify the data for the sheet

When the sheet is placed outside the ForEach, it is not possible to get the specific element in the iteration, so the data passed to the sheet cannot be specified. This problem can be solved by specifying the id of the element as the id of the element itself when iterating.

```swift
@State private var selectedMind: Mind? = nil

ForEach(filterMinds, id: \.id) { mind in
	Text(mind)
		.contextMenu {
			Button(role: .destructive) {
			    removeMind(mind)
			} label: {
			    Label("Delete", systemImage: "trash")
			}
		}
}
.sheet(isPresented: $showEditMindView) {
    EditMindView(mind: selectedMind)
}
```
