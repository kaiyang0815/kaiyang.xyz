## Use UIViewRepresentable create a canvas view

### Create a canvas view

The pencilkit not yet support SwiftUI, so we need to use UIViewRepresentable to create a canvas view.

```swift
import PencilKit
import SwiftUI

struct CanvasView: UIViewRepresentable {
    func makeUIView(context: Context) -> PKCanvasView {
        let canvasView = PKCanvasView()
        canvasView.tool = PKInkingTool(.pen, color: .black)
        return canvasView
    }

    func updateUIView(_ uiView: PKCanvasView, context: Context) {}
}
```

The above code have a PKInkingTool, which is a default tool in PencilKit. We can change it to other tools and colors.

### Create a SwiftUI view

Now, we can create a SwiftUI view to display the canvas view.

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        CanvasView()
            .ignoresSafeArea()
    }
}
```
