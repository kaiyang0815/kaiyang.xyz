## How to add a text in picker

```swift
    Picker("Source", selection: $selectedSource) {
        Text("All")
            .tag(nil as SourceConfig?)
        ForEach(sources, id: \.id) { source in
            Text(source.generalRule.siteName)
                .tag(source)
        }
    }
```
